EPICS JavaIOC: dbDerfinition example
package: org.epics.ioc.dbDefinition.example
2006.05.18

This package is an eclipse JUnit test for org.epics.ioc.dbDefinition. It contains:

  • XMLToDBDTest.java
    A JUnit test for XMLToDBD.
  • test.xml
    An xml file containing database definitions.

This is a JUnit test for DMLToDBD. This also is a test for dbDefinition and pvAccess because XMLToDBD makes extensive use of dbDefinition and pvAccess. It also provides an example of parsing database definitions. When run it produces four warning messages. These demonstrate what happens when an xml file has errors. The output is a dump of all the dsatabase definitions defined by test.xml.
