EPICS JavaIOC db test
package: org.epics.ioc.db.test
2007.01.09

Introduction

This package is an eclipse JUnit test for org.epics.ioc.dbAccess. It contains The following JUnit tests:

  • DefaultTest
    Reads a set of database definition files that have defaults for all fields that are allowed to have defualts. Creates an instance definition which does not set any field values. It dumps the resultig instance.
  • ParentTest
    Reads a set of database definitions and record instances. Show the parent and record for specified fields.
  • AccessTest
    Tests DBAccess.
  • ListenerTest
    Tests DBListener.
  • ReplaceTest
    Test repacing the default implementation of all the field types.
  • SubstituteTest
    Tests include and macro substitution. This also is a test for pv, dbd, and db because XMLToDBD is called, which makes extensive use of dbd and pv, and XMLToIOCDB is called, which makes extensive use of db. It also provides an example of parsing database definitions.

It also has a Java class TestListener, which is used by some of the other tests and is an example of implementing a DBListener.

The examples use the DBD definitions defined in javaIOC/dbd.

The following database definition files are available:

  • protoAiDB.xml
    Used by ReplaceTest.
  • replaceDBD.xml
    Include statements for the files requires by ReplaceTest.
