Local Channel Provider

This package implementats a channel access client for accessing records in the local IOC database. It has the name "local" This is created automatically by ChannelProviderLocalFactory . The factory creates a single instance of an implementation of a channel access client for accessing local records. ChannelProviderLocalFactory has only one public method.

    public class ChannelProviderLocalFactory  {
        static public void register();
    }

This method is called by ChannelFactory when it is called before a local Channel Access is registered.
