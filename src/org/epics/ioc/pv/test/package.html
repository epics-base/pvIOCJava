EPICS JavaIOC: pv test
package: org.epics.ioc.pv.test
2007.01.09

Introduction

This package is an eclipse JUnit test for org.epics.ioc.pv. It contains two class definitions:

  • DatabaseExampleTest.java
    The JUnit test.
  • DatabaseExample.java
    A class that creates a databases containing a field of each possible Type.

DatabaseExampleTest

This has the definition:

public class DatabaseExampleTest extends TestCase {
    public static void testBoolean();
    ...// tests for all the other scalar types
    public static void testStructureAndProperty();
    public static void testBooleanArray();
    ...// tests for arrays of all element types

Each scalar test produces output like:

field boolean type pvBoolean isMutable true 
value true true

Each array test produces output like:

field booleanArray type pvArray isMutable true 
elementType pvBoolean 
value {true false true }

testStructureAndPropertry produces the following:

field value type pvDouble isMutable true 
property{
    {name = displayLimit field = displayLimit}
}
value 0.000000 0.000000

DatabaseExampleTest

This has the definition: 

public class DatabaseExample {
    public DatabaseExample(String name);
    public String getName();
    public PVData createData(String name,Type type, Property[] property)
    public PVEnum createEnumData(String name,
          boolean choicesMutable, Property[] property);
    public PVStructure createStructureData(String name, String structureName,
          PVData[] pvData, Property[] property);
    public PVArray createArrayData(String name,Type type, Property[] property);
}

Thus it creates fields with data of any of the supported types.
