Overview

This package will have support for device abstractions. For now it only has an example foer a poser supply.

Power Supply Support

    public class PowerSupplyFactory {
        public static Support create(DBStructure dbStructure);
    }

This is an example of support for a power supply. Given a voltage and power to computes the current. It supports fields for reading the voltage and for writing the current. The complete set of fields it suppports are:

voltage
This is required. It must be a double structure.
current
This is required. It must be a double structure.
power
This is required. It must be a double structure.
