EPICS JavaIOC: caV3
Channel Access Version 3
package: org.epics.pvioc.caV3
2011.12.19

This package provides:

  1. A caV3 server that provides access to the V4 records in a javaIOC.
  2. A V4 pvAccess server that provides access to records in a V3 IOC.

See documentation/pvIOCJava.html for a more complete overview.
